Multi-Well Salt Solution-Mining Intercept

Kansas, USA  ·  2023 – ongoing

A repeat salt solution-mining program — connecting five-well sets from a single access well via coiled-tubing ranging-while-drilling

The challenge
  • —A regulator required positive well-to-well intersection instead of hydraulic fracturing.
  • —Each set: five vertical salt-solution wells in a line, connected below the casing shoe (~450 ft TVD).
  • —Shallow, high-dogleg geometry into narrow 3–11 ft TVD windows, with no target-well surveys.
  • —The result had to be repeatable across successive well sets, year over year.
Our solution

Torped™ rotating-magnet ranging on a coiled-tubing ranging-while-drilling unit: a single access well lands shallow, then drills the line — ranging to and intercepting each target well below the shoe. The first set pioneered a formation-current-injection AMR-while-drilling field test; composite-survey shift corrections steer each pass-by and intercept.

Repeat program: first set (2023) → third set (2025); a fourth planned for 2026
The result
3 well sets
connected (2023–2025); a 4th planned for 2026
  • ·First-of-its-kind formation-current-injection AMR-while-drilling field test
  • ·Each set links five vertical wells from one access well — no hydraulic fracturing
  • ·Detected and corrected real survey shifts (e.g. +1.7° azimuth; up to ~27 ft) in real time
  • ·Hydraulic communication confirmed; missed wells re-aligned by sidetrack
