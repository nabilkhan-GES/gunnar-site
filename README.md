# Gunnar Energy Services — Website

Production website (Astro). Source lives in `web/`.

```bash
cd web && npm install && npm run dev     # local preview at http://localhost:4321
npm run build                            # static output → web/dist
```

The live site is published to GitHub Pages from the `gh-pages` branch
(Settings → Pages → Deploy from a branch → gh-pages → /root).
See `web/README.md` for full details and the pre-launch checklist.
