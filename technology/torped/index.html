At-Bit Active Magnetic Ranging

Torped™

Real-time ranging data at the bit — vendor-neutral, and deployable in virtually any BHA.

Talk to a ranging engineer All technology →
How it works

A magnetic bit sub installed between the bit and the motor/RSS that generates a field detectable by Torped's wireline receiver in the offset well — maximum directional control for intersect or collision-avoidance, with no MWD/RSS integration required.

The Torped™ magnetic bit sub generates a field that doesn't interfere with the MWD but is readily detected by the Torped™ receiver in the offset wellbore — putting ranging information where you need it, at the bit, for maximum directional drilling control.

It requires no electrical or communications integration with any downhole MWD or RSS, so it deploys seamlessly with the operator's chosen directional driller. MWD/DD professionals of any vendor can be trained to run it independently — no extra personnel on location — and custom magnet sub sizes fit virtually any BHA.

A gyro module shares the sensor protocol and runs on mono-conductor wireline for vertical cased-hole intercepts — an industry first — and proprietary software generates a new intersect-or-avoid well plan after each shot.
