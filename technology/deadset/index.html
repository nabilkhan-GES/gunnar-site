A no-access wireline active ranging system built from the ground up for first-time intersection.

How it works

An all-new MEMS / solid-state active ranging intercept system with interchangeable sensor packages and an optional gyro for vertical intersects — engineered for the highest probability of first-time intersection on the market.

DeadSet™ injects current to excite the target casing and measures the field to compute distance and direction. Built from the ground up on a MEMS / solid-state platform, it delivers precise, repeatable results with a ruggedness and simplicity legacy architectures can't match — without sacrificing precision or speed of service.

Its completely modular design lets sensor packages be interchanged for each stage of the intercept, and proprietary software interprets the ranging data and generates a new intersect well plan after every shot — reducing the on-site personnel usually needed for interpretation.

An optional gyro module shares the sensor communications protocol, so a magnetic/gyro assembly runs on a single trip. Gyro compassing at vertical orientations removes the ambiguity common to magnetic-only systems; when it isn't needed, the module unscrews to minimize downhole electronics and cost.
